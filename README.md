# Herbário do Victor

Um herbário digital que combina a precisão da engenharia com a organicidade da botânica. Este projeto é uma aplicação web progressiva (PWA) construída com React, TypeScript e GSAP para criar uma experiência de usuário imersiva e editorial.

## Tecnologias

- **React 18**: Biblioteca para construção da interface.
- **TypeScript**: Tipagem estática para robustez do código.
- **Vite**: Build tool ultrarrápido.
- **Tailwind CSS**: Framework de utilitários CSS para estilização.
- **GSAP (GreenSock)**: Biblioteca de animação para transições complexas e interações.
- **Lenis**: Scroll suave para uma experiência de navegação premium.
- **Lucide React**: Ícones SVG leves e consistentes.

## Funcionalidades

- **Design Editorial**: Layout inspirado em revistas de design, com tipografia forte e grids precisos.
- **Scroll Suave**: Navegação fluida com Lenis e barra de progresso de leitura.
- **PWA**: Instalável em dispositivos móveis com suporte offline.
- **Mobile First**: Interface otimizada para toque com "Thumb Zone" navigation.
- **Animações**: Transições de página e micro-interações refinadas.

## Como Executar

1. Clone o repositório.
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
4. Abra `http://localhost:3000` no navegador.

## Estrutura do Projeto

- `/src/components`: Componentes reutilizáveis (UI, Layout).
- `/src/pages`: Páginas principais da aplicação.
- `/src/lib`: Configurações de bibliotecas (GSAP, Lenis).
- `/public`: Assets estáticos.
